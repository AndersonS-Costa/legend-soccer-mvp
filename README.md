# Legend Soccer MVP

## Overview
Legend Soccer MVP is a project designed to engage soccer fans by allowing them to vote for their most valuable player (MVP) in a given match. The application tracks votes, displays statistics for players, and provides insights into voting trends over time. This project aims to enhance fan interaction and foster community engagement around soccer events.

## Features
- **MVP Voting**: Users can vote for their favorite players after matches.
- **Player Statistics**: View individual player statistics including goals, assists, and total votes.
- **Voting Trends**: Analyze voting patterns and trends across different matches and players.
- **User Profiles**: Users can create profiles to track their voting history and favorite players.
- **Live Updates**: Real-time updates on match outcomes and player performances.

## Tech Stack
- **Frontend**: React.js for building user interfaces.
- **Backend**: Node.js with Express for server-side logic.
- **Database**: MongoDB for storing user data and votes.
- **Authentication**: JSON Web Tokens (JWT) for secure user authentication.
- **Deployment**: Deployed on Heroku for web hosting and continuous integration.

## Roadmap

3. **Phase 3**: Introduce voting trends and analytics.
4. **Phase 4**: Enhance UI/UX for better user experiences.
5. **Phase 5**: Add mobile support and native application development.

## Setup Instructions
To set up the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/legend-soccer-mvp.git
   cd legend-soccer-mvp
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables in a `.env` file based on the `.env.example` provided.
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:3000` to see the application in action.
